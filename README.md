Supabase: products table
========================

Run this SQL in Supabase SQL Editor to create the `products` table:

```
CREATE TABLE products (
	id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
	title VARCHAR(255) NOT NULL,
	price NUMERIC DEFAULT 0,
	image TEXT,
	created_at TIMESTAMP WITH TIME ZONE DEFAULT now()
);
```

Notes:
- After creating the table you can either use the seed button in the app (if your anon key permits insert) or insert rows via the Table Editor.
- Ensure your RLS/policies allow public insert/select if you use anon key; otherwise use a server-side endpoint or service role key securely.
